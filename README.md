# Book A Doctor

Book A Doctor is a full-stack healthcare appointment booking application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js).

The application helps patients find doctors, book appointments, and manage their healthcare interactions online. It also provides separate dashboards for doctors and administrators to manage appointments and system activities.

## Features

* User Registration and Login
* Secure Authentication using JWT
* Password Encryption using bcrypt
* Doctor Browsing and Profile Viewing
* Appointment Booking and Management
* Patient Dashboard
* Doctor Dashboard
* Admin Dashboard
* Doctor Approval System
* Medical Report Upload
* MongoDB Database Integration
* Responsive User Interface

## Technologies Used

### Frontend

* React.js
* Bootstrap
* Axios
* React Router

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Security

* JWT Authentication
* bcrypt Password Hashing

## Project Modules

### Patient Module

Patients can register, log in, view doctors, book appointments, and track appointment status.

### Doctor Module

Doctors can manage appointment requests by approving, rejecting, or completing appointments.

### Admin Module

Administrators can monitor users, doctors, appointments, and approve doctor registration requests.

## Outcome

This project demonstrates full-stack web development concepts including authentication, CRUD operations, role-based access control, API integration, and database management using the MERN stack.
